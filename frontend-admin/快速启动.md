# 🚀 快速启动指南

## 第一步：安装依赖

```bash
npm install --legacy-peer-deps
```

## 第二步：启动开发服务器

```bash
npm run dev
```

## 第三步：访问系统

打开浏览器访问：**http://localhost:5174**

## 第四步：登录管理后台

- **默认账号：** `admin`
- **默认密码：** `admin123`

---

## ⚠️ 注意事项

1. **后端服务：** 确保后端 API 服务已启动（默认地址：http://localhost:3000）
2. **Node.js 版本：** 需要 Node.js >= 18
3. **端口占用：** 如果 5174 端口被占用，可以修改 `vite.config.ts` 中的端口配置

---

## 🎯 主要功能

- ✅ 用户管理（创建、编辑、调整积分、重置密码）
- ✅ 开奖历史（查看、搜索、同步数据）
