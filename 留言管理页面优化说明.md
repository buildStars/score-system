### 2️⃣ 移动端适配

#### 响应式布局
```scss
@media (max-width: 768px) {
  // 页面内边距调整
  .message-list-page {
    padding: 10px;
  }

  // 工具栏垂直布局
  .toolbar .search-box {
    flex-direction: column;
    align-items: stretch;
  }

  // 表格字体缩小
  .table-wrapper :deep(.el-table) {
    font-size: 12px;
  }

  // 分页简化（隐藏页面大小选择和跳转）
  :deep(.el-pagination) {
    .el-pagination__sizes,
    .el-pagination__jump {
      display: none;
    }
  }
}
```

#### 对话框适配
- ✅ 宽度改为 `90%`，最大宽度 `600px`
- ✅ 自动适应屏幕尺寸
- ✅ 查看详情对话框使用 `label-width="80px"` 减少标签宽度
- ✅ 回复对话框优化为 Alert 提示框展示原始留言

### 3️⃣ 精简优化

#### 表格列精简
**优化前：**
- ID（80px）
- 姓名（120px）
- 联系方式（180px）
- 留言内容（250px）
- 状态（100px）
- 提交时间（180px）
- 操作（200px）

**优化后：**
- 姓名（100px）✅ 缩小
- 联系方式（140px）✅ 缩小
- 留言内容（200px）✅ 缩小
- 状态（80px）✅ 缩小
- 提交时间（155px）✅ 缩小
- 操作（180px）✅ 精简

#### 操作按钮精简
**优化前：**
```html
<el-button link type="primary">
  <el-icon><View /></el-icon>
  查看
</el-button>
```

**优化后：**
```html
<el-button link type="primary" size="small">
  查看
</el-button>
```
移除图标，只保留文字，更简洁

#### 回复对话框精简
**优化前：**
- 使用独立的 div 容器展示原始留言
- 样式较复杂

**优化后：**
- 使用 `el-alert` 组件展示原始留言
- 样式更统一，代码更简洁
