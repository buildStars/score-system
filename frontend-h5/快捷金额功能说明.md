## 🎨 交互效果

### 1. 默认状态
- 所有按钮为灰色（default）
- 均可点击

### 2. 选中状态
- 已选金额的按钮变为蓝色（primary）
- 高亮显示当前选择

### 3. 点击效果
```typescript

  ↓

  ↓
手续费自动计算并更新
```

## 💡 使用场景

### 场景1：快速下注
```
1. 用户打开首页
2. 选择"倍数下注"
3. 点击 [1000] 快捷按钮
4. 自动填充倍数为 1000
5. 确认下注
```

### 场景2：切换金额
```
1. 点击 [500] - 金额变为 500
2. 点击 [1000] - 金额变为 1000
3. 点击 [5000] - 金额变为 5000
```

### 场景3：手动输入
```
1. 点击快捷金额 [1000]
2. 不满意，手动修改为 1200
3. 继续下注
```

### 2. 设置金额函数

```typescript
/**
 * 设置快捷金额
 */
const setQuickAmount = (type: string, amount: number) => {
  if (type === 'multiple') {
    multipleBet.amount = String(amount)
  } else {
    comboBet.amount = String(amount)
  }
}
```

**功能**：
- 点击快捷金额按钮
- 自动填充到对应的输入框
- 手续费自动重新计算

## 🎯 使用说明

### 倍数下注
1. 进入首页
2. 选择"倍数下注"tab
3. 点击快捷金额按钮（如：1000）
4. 系统自动填充倍数为 1000
5. 查看手续费（自动计算为 30）
6. 点击"确认下注"

### 组合下注
1. 进入首页
2. 选择"组合下注"tab
3. 先选择组合（如：大）
4. 点击快捷金额按钮（如：500）
5. 系统自动填充金额为 500
6. 查看手续费（自动计算为 25）
7. 点击"确认下注"
